# Color Clock (React + Vite)

A simple dynamic digital clock built with React and npm.  
It displays the current date and time and updates every second.

## Technologies Used
- React
- Vite
- npm
- date-fns

## How to Run Locally

1. Clone the repository:
   git clone https://github.com/your-username/color-clock.git

2. Navigate into the project:
   cd color-clock

3. Install dependencies:
   npm install

4. Start the development server:
   npm run dev

5. Open your browser at:
   http://localhost:5175/ (This is my current browser)

## Features
- Live updating digital clock
- Formatted date and time using date-fns
- Styled with CSS
